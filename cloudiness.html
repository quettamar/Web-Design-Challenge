Cloudiness

For cloudiness versus latitude, there is no correlation. At any given latitude, cloudiness data covers the full available range. For example, the equator includes data from areas like Brazil and Indonesia which get a lot of rainfall, therefore, more clouds. There are also areas which include deserts that don't have a lot of clouds.
